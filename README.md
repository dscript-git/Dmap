Dmap is an automation script for Nmap. It has most of
the needed network scanning options you may need.
Ideal for System Admins and I.T. professionals.
Tested on Linux and Android Termux app.

Prerequisites:
Linux OS
Nmap
Python 3.x

Usage:
python dmap.py
